# CLI Contract Layer (Alpha Tier)

This folder contains the **contract-first** building blocks required by the Lab CLI MVP contract:

- **Intent registry** (`intents.ts`): stable intent identifiers + metadata
- **Schema layer** (`schema.ts`): versioned Zod schemas for JSON output
- **Envelope builders** (`envelope.ts`): canonical constructors for contract outputs
- **Exit codes** (`exitCodes.ts`): stable, predictable meanings
- **Capabilities** (`capabilities.ts`): machine-discoverable CLI support surface

Source of truth: **Lab CLI MVP Contract (v0.x)**.
